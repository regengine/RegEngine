## Input Variables

### Facility Profile
- **Facility Type**: Research / Fuel Cycle / Power Reactor (e.g., Power Reactor)
- **Annual Inspections**: NRC inspection frequency (e.g., 12)
- **CAP Items**: Corrective Action Program items/year (e.g., 500)
- **Safety Culture Surveys**: Annual survey count (e.g., 2)

### Current Compliance Costs
- **NRC Inspection Prep**: Hours preparing for inspections (e.g., 2,000 hrs/year)
- **Blended Hourly Rate**: Nuclear engineer cost (e.g., $100/hr)
- **CAP Management**: Hours managing corrective actions (e.g., 3,000 hrs/year)
- **Evidence Retrieval**: Hours finding historical records (e.g., 500 hrs/year)
- **NRC Findings**: Annual cost of violations (e.g., $200K)
