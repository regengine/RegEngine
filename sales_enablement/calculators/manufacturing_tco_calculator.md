## Input Variables

### Company Profile
- **ISO Certifications**: Which standards? (9001, 14001, 45001 - e.g., 3)
- **Annual NCRs**: Non-conformance reports per year (e.g., 120)
- **Annual CAPAs**: Corrective actions created (e.g., 200)
- **Employees**: Total headcount (e.g., 500)
- **Annual Revenue**: Total manufacturing revenue (e.g., $75M)

### Current Compliance Costs
- **ISO Audit Prep**: Combined prep for all 3 standards (e.g., $250K/year)
- **CAPA Tracking Labor**: Hours managing CAPAs (e.g., 2,000 hrs/year)
- **Blended Hourly Rate**: Quality manager cost (e.g., $70/hr)
- **Supplier 8D Review**: Hours reviewing supplier quality (e.g., 600 hrs/year)
- **Audit Finding Follow-Up**: Hours closing audit findings (e.g., 400 hrs/year)
- **CAPA Closure Rate**: % of CAPAs closed on time (e.g., 60%)
