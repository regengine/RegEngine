# RegEngine Production Bundle

This bundle contains essential documentation, operational tools, and developer resources for the production deployment and compliance readiness of RegEngine.

## Directory Structure

- `documentation/`: Compliance mappings, whitepapers, and crosswalks.
    - `Investor_Whitepaper_v1.0.md`
    - `SOC2_Type1_Mapping.csv`
    - `HIPAA_Subpart_E_Crosswalk.csv`
- `operational_tools/`: Regression tests, dashboard templates, and onboarding guides.
    - `RLS_Regression_Tests.yaml`
    - `Compliance_Drift_Dashboard_Template.json`
    - `Agent_Onboarding_Sim_Pack.txt`
- `developer_resources/`: SQL snippets, Terraform config, and utility scripts.
    - `SQL_Policy_Builder.sql`
    - `Config_as_Code_Snippets.tf`
    - `Audit_Trail_Replayer.py`

## Usage

This bundle is designed to support security audits, compliance verification, and operational readiness for RegEngine enterprise deployments.
